# Sanctum MCP plugin directory (Broca repo)

Set **`MCP_PLUGINS_DIR`** to this directory when running [sanctumos/smcp](https://github.com/sanctumos/smcp). SMCP loads each immediate child folder that contains a **`cli.py`**.

| Subdir | Purpose |
|--------|---------|
| **`broca/`** | Tools `broca__*` — wraps `python -m cli.*` for this stack. See **`broca/README.md`**. |
